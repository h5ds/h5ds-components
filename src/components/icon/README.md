## Props

| 参数 | 说明 | 类型 | 必须参数 | 默认值 |
| ---- | ---- | ------- | -------- | ----------- |
| type | 图标类型，对应 [https://fontawesome.com/v4.7.0/icons/](https://fontawesome.com/v4.7.0/icons/) 的类型，如： 500px\angular | string | 是 | - |
| spin | 是否旋转 | boolean | 否 | false |
| style | 图标自定义样式，如 `{ fontSize: 50, color: '#666' }` | object | 否 | {} |
